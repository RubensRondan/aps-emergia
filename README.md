# Sistema de Cálculo Emergético

Sistema desenvolvido em Python para realização de cálculos emergéticos simplificados aplicados à análise ambiental.

## Funcionalidades

- Importação CSV
- Cálculo emergético
- Geração de gráficos
- Relatórios PDF
- Banco SQLite

## Tecnologias

- Python
- PyQt5
- Pandas
- SQLite
- Matplotlib
- ReportLab
